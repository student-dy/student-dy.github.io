5. Project Ideas

Overview

Weather based smart alarm application- The majority of students and worker, including me, take various kinds of vehicles to reach their own place. If you unexpectedly find your window being splashed with rain or snow in the morning, you would most likely be late for not being able to ride your bike or traffic congestion. It would be great if all of us had thoroughness to check the weather of the next day before going to sleep, but it usually doesn’t work that way. Therefore, I propose an alarm that smartly sets its alarm time differently from the originally set time depending on weather and user customization.

Motivation

Schedule depends on many factors. One of the crucial factors is weather. To verify whether it’s a good day to go on a picnic or amusement park the next day, most of the people check the weather of the D-day. Besides these special occasions, however, not much portion of the people check on weather of the ordinary days. Also, the weather forecast might change while asleep. I have finally found a way to keep us being lazy and still be able to get on time even under an unexpected weather.

Description

Brief Description

  • This product is an extension of basic alarm application on smartphone. Different from other alarm apps, which can only ring an alarm at exact time, the new application provides weather based alarm that can ring at different times when weather changes. Not only time changes, but display of the alarm is also another distinctive feature. To let an user know the weather at a glance, each unique display is provided.

This application mainly contains Alarm Managing System and Weather Managing System:

  • Alarm Management System controls alarm information and rings alarms.
  • Weather Managing System gets weather forecasting information.

Location properties

  • User can register his/her current location using either GPS or manual input.
  • Weather information will be based on the set location.

The application provides several options for customization of alarms:

  • Alarm with Weather Dependency option (Rain or Snow)
  • Ringing alarms N minutes earlier/ later/ never in case of rain or snow.

To develop the application, I have to focus on these functionalities.

  • App provides stable basic alarm: At the exact user wanted time.
  • Response time of all functionalities should be less than 2 seconds.

To run the application, operation environment is expected as follows:

  • Android OS or IOS (to be decided)
  • Internet connection for getting weather information

Assumptions and Dependencies

  • Assume that Google weather service provides exact information.
  • Assume that people need more/less time at morning depends on weather.

Tools and Technologies

Since this application is a smartphone application, it requires an Android OS or a device that supports IOS. In addition, GPS and alarm function for obtaining positional information are needed. An Internet connection is also required to receive weather information. To develop, the Android Studio Tool or the Xcode Platform is needed depending on the operation environment of the target device.

Skills Required

In order to operate on a device supporting Android OS, it is necessary to develop the application on the Android Studio Tool based on Linux OS and Java language. Besides, to operate on IOS device, development on Xcode Platform based on Mac OS Is needed. On the front-end side, it is need to create the icons and pictures and place them in the proper places so that users can know each function intuitively. Also, it should be able to improve the problem through testing. It is a good idea to classify and document functional and nonfunctional requirements as needed so that development of application can be done systematically.

Outcome

The final output of this project will be in application form. This application provides an alarm clock which can be used to set the alarm time. With this application, we will be free from lateness and schedule fluctuations that we had to accept because of the weather before, and we can lead a better life. In addition, during the course of this project, I can learn skills and abilities as a developer while experiencing the software development phases.
